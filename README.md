# theater-ticket-system
A python GUI application for theater ticket purchasing and sales tracking

## Features
- Account creation and login
- Ticket selection and purchasing
- Manager-only password update
- Manager-only sales viewing
- Daily sales reporting
- File-based ticket/seat tracking

## Technologies
- Python
- Tkinter
- File I/O

## Note
This was built as an academic project and uses local file storage for learning purposes
